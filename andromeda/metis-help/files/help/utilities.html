METIS utilities keybindings

Some of the useful default bindings that Metis Linux has made for you.

Neovim bindings

  • n : Normal
  • t : Terminal
  • i : Insert


    Keybind            Action
i + Ctrl-h Move left in insert mode
i + Ctrl-j Move down in insert mode
i + Ctrl-k Move up in insert mode
i + Ctrl-l Move right in insert mode
n + Ctrl-s Save the file
n + Ctrl-q Close the current buffer
n + TAB Switch to the next buffer in the bufferline
n + Shift-TAB Switch to the previous buffer in the bufferline
n + leader + a Toggle NvimTree
i + jj Exit insert mode
n + J Move down and center the screen
n + K Move up and center the screen
n + leader + v Vertical split
n + leader + h Horizontal split
n + Ctrl-h Move to the left split
n + Ctrl-j Move to the bottom split
n + Ctrl-k Move to the top split
n + Ctrl-l Move to the right split
n + leader + x Close the current split

TMUX bindings

  • C-a : prefix


Keybind Action
prefix + - Horizontal split
prefix + | Split window horizontally
prefix + < Export default clipboard
prefix + h (left key) Select pane to the left
prefix + j (down key) Select pane below
prefix + k (up key) Select pane above
prefix + l (right key) Select pane to the right

Terminal bindings

METIS uses suckless's terminal which is very minimal and yet powerful. To enhance the overall workflow the followings are default bindings that we wrote for you.



Keybind Action
A + c Copy selection
A + v Paste clipboard
A + l Shows all urls of terminal in dmenu
A + o Copy the output of command using dmenu

zsh shell bindings

The zsh shell of metislinux has vim bindings, which means you can get all those bindings of vim directly in your shell to navigate faster..



Keybind Action
ESC Toggle insert mode
C + 0 Go to start of a command
w Go ahead to a word
b Go back of a word
S + a Go to end of the line
l go ahead one letter
h go back one letter
k cycle history up
j cycle history down
C + f Open file tree with fzf and bat

These are the pretty common bindings, 3w, 5b or any would even work :)
