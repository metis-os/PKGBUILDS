How to fix invalid or corrupted package (PGP signature)issue?

  • Open a terminal
  • Run: sudo pacman-key --init
  • Then Run: sudo pacman-key --populate
  • Done! (Also delete the corrupted pkgs too.)

What is Runit?

Runit is a simple and lightweight init system for Unix-like operating systems. It is designed to be easy to understand, reliable, and efficient. Runit follows the Unix philosophy of doing one thing well.

Runit manages system services, provides process supervision, and handles system startup and shutdown. It is known for its simplicity, stability, and fast boot times.

Enabling, Disabling, and Restarting Services

To enable a service in Runit, create a symbolic link from its service directory to the /etc/runit/sv/ directory. For example:

ln -s /etc/runit/sv/service_name /run/runit/service/service_name

To disable a service, remove the symbolic link from the /etc/runit/sv/ directory:

rm /run/runit/service/service_name

To start/restart/stop a service, send the SIGTERM signal to its main process using the sv command:

sv < start/stop/restart > service_name

Replace service_name with the actual name of the service.
